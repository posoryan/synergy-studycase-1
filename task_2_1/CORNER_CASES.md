# Граничные случаи — задача 2.1

Правило: первый индекс max и первый индекс min, сумма отрицательных строго между ними.

## Размер массива

- пустой или один элемент -> 0
- два элемента -> max и min соседи, между ними пусто -> 0

## Положение max и min

- max слева: `[8, -1, -2, -3, 2]` -> -3
- между только неотрицательные: `[1, 5, 2, 9, 3]` -> 0
- max и min соседи: `[3, -1, -2, 7, -4, 1]` -> 0
- min слева, max справа: `[-5, 2, 3, 10]` -> 0

## Состав промежутка

- несколько отрицательных: `[20, -3, -5, -7, 10]` -> -8
- ноль не отрицательный: `[5, 0, 0, -3, 1]` -> 0
- все одинаковые: `[4, 4, 4]` -> 0
- дубликат max (первое вхождение): `[9, -1, -2, 1, 5, 9]` -> -1
- дроби: `[5.0, -1.2, 2.0, -4.1]` -> -1.2

## Ошибки ввода (режим input())

- нечисловой N или элементы
- N <= 0
- количество чисел не совпало с N

Проверяются вручную через `python sum_negative_between_extremes.py`.
